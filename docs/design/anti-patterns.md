# Chatterbrain UI Anti-Patterns

Avoid these patterns unless explicitly requested:

## Generic AI SaaS look

- Gradient headlines
- Glass panels
- Floating bento cards
- Neon glows
- Abstract blobs
- “AI magic” sparkles everywhere

## Over-carded layout

- Every section inside a rounded rectangle
- Same visual weight for primary and secondary information
- Excessive shadows
- Dashboard template feeling

## Color overload

- Different colored icons for every section
- Multiple competing accent colors
- Purple, blue, green, orange all on the same screen
- Semantic colors used decoratively

## Childish neurodivergent design

- Excessive gamification visuals

## Bland minimalism

- Too much whitespace with no hierarchy
- Weak title/action relationship

## Corrective rule

When revising UI, do not simply make it more minimal. Make it more intentional.
